## 评分重点

- 上层料盘：10 分。
- 下层料盘：20 分。
- 料盘需要从货架取出并放到 0.2m x 0.2m 出库区域。

## 第一阶段目标

先做上层料盘，不要同时追上下层。

```text
识别上层料盘 -> 得到 camera_xyz -> 固定动作接近 -> 抓取 / 托出 -> 放到出库区附近
```

## 分工

闫帅辰：视觉输出。

```text
RGB/depth/camera_info -> bbox -> center_pixel -> depth -> camera_xyz -> JSON
```

王森：动作骨架。

```text
手臂轨迹 -> 夹爪开合 -> 回撤 -> 接视觉点
```

## Day2 验收

- 上层料盘有 JSON。
- 动作骨架能动手臂和夹爪。
- 有截图或视频。
- 有失败原因。
