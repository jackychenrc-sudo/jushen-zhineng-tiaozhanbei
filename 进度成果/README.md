# 进度成果

这个目录只放每天同步用的小成果，方便晚上汇总进度。

## 可以放什么

```text
小截图：jpg / png
小 JSON：视觉输出、检测结果、坐标结果
小日志：txt / md
当天结果说明：md
```

## 不要放什么

```text
rosbag
大视频
大模型权重
build / devel / log
完整数据集
```

大文件放服务器路径或网盘，只在 `文档/测试记录.md` 里写清楚位置。

## 建议放法

按日期和场景分目录：

```text
进度成果/
└── 2026-07-12/
    ├── scene1/
    ├── scene2/
    ├── scene3/
    └── 测试汇总.md
```

文件名尽量写清楚是谁、什么内容：

```text
闫帅辰_scene3_上层料盘检测.jpg
闫帅辰_scene3_上层料盘3D.json
王森_scene3_夹爪测试说明.md
王仔俊_scene1_包裹截图.jpg
史智涛_scene2_零件分类结果.json
窦欣悦_当天测试汇总.md
```

## 最重要

上传成果时，同时在 `文档/测试记录.md` 或当天汇总里写一句：

```text
这个文件是什么？
对应哪个 scene 和 seed？
成功了什么？
失败或卡住了什么？
```
